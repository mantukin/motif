Motif is an AI-assisted MIDI composition and sequencing app for sketching musical ideas before finishing them in a DAW such as Reaper, Ableton Live, or Logic Pro.

Live app: [mantukin.github.io/motif](https://mantukin.github.io/motif/)

## What You Can Do

- Compose melodies, chords, and arrangements in a piano roll editor.
- Build drum parts in a dedicated step sequencer.
- Write lyric-aware vocal lines in the voice melody editor.
- Follow vocal phrasing with the built-in karaoke strip.
- Generate musical ideas with AI-assisted composition tools.
- Import audio and convert it to MIDI with Basic Pitch.
- Export projects or clips for use in your DAW.

## About This Repository

This public repository hosts the deployed web build and static assets for GitHub Pages.

The main development source code is maintained separately in a private repository. This repo exists to publish the live web version of Motif.
